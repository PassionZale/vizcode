## a-rol

10 月初, 订阅了智普的编程套餐, 深度使用了两个月.

在交流群里面, 大家问的最多的就是 lite 套餐如何使用视觉理解.

## begin

智普的编程套餐总共分三档,

168 一年的 lite, 840 一年的 pro, 1680 一年的 max.

## scene_1

lite 对比 pro, 便宜了 672, 同时也缺少了视觉理解/联网搜索/网页读取三个功能,

我们可以使用这三个 mcp 来平替它们:

视觉理解用 智普官方的 z ai mcpserver,

联网搜索用 tavily mcp,

网页读取用 mcp fetch server.

## scene_2

智普官方的 mcp 并不是只能用他们家的模型哈,

注册模搭社区并且绑定阿里云账号, 每天就有 2000 次的免费调用,

这里我们将 mcp 环境变量中的 base url 和 vision model 替换成模搭和千问,

试一下效果, 将一个本地的图片路径丢给 claude, 让它描述一下内容,

mcp 的工具被调用, 稍等一下, 就能看到图片的解析结果了.

## scene_3

联网搜索, 需要在 tavily 上面注册一个账号, 每个月就有 1000 次的免费调用,

试一下效果, 让 claude 给我生成一份智普的模型报告,

mcp 的工具被调用了两次, 然后就能看到报告了.

## scene_4

网页读取, 这个 fetch mcp 呢, 有三种工具, 可以将网页转成 markdown html json,

这里我们随便试一下, 丢一个网址让 claude 进行总结.

工具被调用, 很快, 总结完成.
