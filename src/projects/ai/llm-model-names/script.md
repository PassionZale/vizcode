**主持人**：其实这些名字背后藏着简单的规律！今天用90秒让你彻底看懂LLM模型命名，从此选模型不再迷茫！

---

## 【核心内容】(0:15-1:15)

**主持人**：首先记住这个万能公式：**品牌名 + 版本号 + 参数大小 + 用途标识**

**[屏幕动画：分解 Llama-3.2-8B-Instruct]**

**主持人**：来看第一个例子 - **Llama-3.2-8B-Instruct**

- **Llama** = 品牌名（Meta公司的模型系列）
- **3.2** = 版本号（第3代第2版本）
- **8B** = 参数大小（80亿个参数，B=十亿）
- **Instruct** = 用途（专门对话的指令模型）

**主持人**：参数大小最关键！直接决定你需要什么硬件：

**[图表对比]**
- **7B** ≈ 手机就能跑
- **8B-13B** ≈ 普通笔记本GPU
- **70B** ≈ 需要专业显卡
- **405B** ≈ 需要10+张高端显卡

---

## 【常见类型】(1:15-1:35)

**主持人**：记住这几个常见标识，你就懂了大半：

**[快速切换的标签动画]**
- **Base** = 基础模型（开发者用的，新手别碰）
- **Instruct/Chat** = 对话模型（聊天的首选！）
- **Vision** = 视觉模型（能看图片的AI）
- **Code** = 代码模型（写代码的助手）
- **Guard** = 守护模型（内容安全的保镖）

**主持人**：还有这些优化版本：
- **FP8/INT8** = 压缩版（省内存但精度略降）
- **Distill** = 蒸馏版（小而快的学生模型）
- **MoE** = 专家混合（智能选择能力模块）

---

## 【实用技巧】(1:35-1:50)

**主持人**：给你三个实用技巧：

**技巧一：看名字选用途**
- 想聊天→找 **Instruct**
- 想看图→找 **Vision**
- 想编程→找 **Code**

**技巧二：看参数选硬件**
- **7-13B** = 新手友好，配置要求低
- **30-70B** = 中级用户，需要好显卡
- **100B+** = 专业级，别轻易尝试

**技巧三：新手入门建议**
从 **Llama-3.2-8B-Instruct** 或 **Qwen-7B-Chat** 开始，配置要求低，功能完整！
