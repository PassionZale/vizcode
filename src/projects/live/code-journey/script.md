十年前我写下了人生中的第一行代码，
我的代码没有改变这个世界，
但是编程却改变了我的生活。

09年，大一，舅舅送了我一台笔记本电脑，
2014年，我用它写下了第一行代码，
用 Java 在控制台打印一个 Hello World。

SpringMVC、Tomcat、Oracle，
这套技术栈让我入门了编程。

2015年，我从武汉去了深圳，开始深漂，
期间结实了超多的朋友。
虽然现在联系的都不多，但是仍然很怀念他们。

工作的这些年我接触了超多的技术栈，
从最开始的 LNMP，
到 Webpack + Vuejs 的 SPA，
再到 Docker、Vite、React、Nextjs 等等。

我维护着 Github，用这些技术栈写了很多好玩的 Side Project：
基于 Astro 的博客；
基于 Slidev 的演讲；
基于 Vuejs 的组件库；
基于 Typescript 的类型体操；
基于 Nodejs 的脚手架；
还有基于 Nextjs 的全栈项目；

我觉得，写代码的这十年，非常的难，也非常的酷。

Hard Coding，
But，
Happy Coding！
